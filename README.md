# EzCast
<b>Description:</b>

DLNA cast for Windows, Linux and MacOS.

# Capabilities
* Ability to stream from local video files.
* Ability to stream from a video URL.
* Ability to find DLNA devices on the network.
* Save and Load list of found devices.

# API endpoints
* "/" - root page.
* "/video.mp4" - url of local video
* "/AVTransport/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/scpd.xml" - AV Transport file
* "/AVTransport/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/control.xml" - AVT Control config xml
* "/AVTransport/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/event.xml" - AVT Event config xml
* "/ConnectionManager/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/scpd.xml" - Connection manager xml
* "/ConnectionManager/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/control.xml" - Connection Manager control config xml
* "/ConnectionManager/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/event.xml" - Connection Manager event config xml
* "/RenderingControl/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/scpd.xml" - Rendering Control xml
* "/RenderingControl/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/control.xml" - Rendering Control config xml
* "/RenderingControl/3e1388f0-1dd2-11b2-83b7-4c0fc7209ecf/event.xml" - Rendering Control event xml

<b>Notes:</b>

- Put the lib folder next to the executable
